Interpretation

The final system combines Transformer-based probability forecasts with a more conservative execution layer. Compared with the earlier frozen configuration, this optimized execution reduces turnover, improves validation drawdown control, and improves the final hold-out test Sharpe and cumulative return. The result should still be interpreted as a regime-sensitive, weak-predictive but economically tradable system rather than a universally stable forecasting engine.
